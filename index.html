Installation Resources

All Platforms

Alternatives for Mac

These are alternatives to the git-scm.com recommendations.

  • Homebrew 
    brew install git
  • Mac Ports 
    port install git-core

Alternatives for Windows

These are alternatives to the git-scm.com recommendations.

  • msysgit - Git for Windows

How to get help

  • Manual Page help
    • man git-[command]
man git-status
  • Short Help
    • git [command] --help
git status --help

Git Tutorials

These are the resources peers, co-workers and myself have found useful. I highly recommend following the directions and the order of this list when reviewing these references.

You might want to grab the Git Quick Reference from the Quick Cheats list below and print it out while reviewing these tutorials.

  1. Git Tutorial - Read from the beginning to the end of Section 4. Don't try to understand it all just read through.
  2. Pro Git - Read from the beginning to the end of Chapter 3. This is where you should try to comprehend what's happening.
  3. Git Immersion - With the background knowledge from above you should start getting hands on with Git now.
